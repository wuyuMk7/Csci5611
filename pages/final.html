Description
The topic of our final project is Tower Defense Game. We implemented it using C++ with OpenGL, along with SDL2, SDL2_mixer, and freetype. We mainly focus on the mechanism of the game and enable players to move the obstacle in the buidling process. This is sort of similart to allow players to change map properties and it will also affect the route for the enemies to move to the target. Since we have written a final project report for our game, we will not provide many details on this page. We have attachecd the link to our report on this page so that you can directly go to check our report.
