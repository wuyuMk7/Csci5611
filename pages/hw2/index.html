Features
  • (5) Realtime rendering.
  • (5) 3D rendering, with user-controlled camera.
  • (5) Realtime user interaction with system.
  • (10) 2D Mass-spring cloth simulation.
  • (10) 3D Mass-spring cloth simulation.
  • (10) Drag-terms. (A wind from the back of the flag keeps blowing the flag.)
  • (10) 1-way cloth-object interaction.
  • (5) 15x15 cloth with fast, smooth motion at 20 FPS.
  • (5) 20x20 cloth at 30 FPS.
  • (5) 30x30 cloth with obstacle interaction at 30 FPS.
  • (5) Eulerian (1st order) (Used in cloth and sound simulation).
  • (5) Higher-oder Explicit (Midpoint used in shallow water simulation).
  • (10) Textured simulated objects. (Textured flag in cloth simulation)
  • (10) Two-way coupling object-simulation coupling. In the cloth simulation, I demonstrated when the cube moves towards the cloth, the cloth will be pushed by the cube. When the cube stands under the cloth, the cloth may be stay on the surface of the cube for a few seconds but may also leave finally due to the gravity or the wind from the back of the flag.
  • (5) Billowing/blowing wind effect simulated with aerodynamic drag (The wind from the back of the flag.)
  • (10) 1D water surface, with shallow water equation.
  • (30) Physical simulated (polyphonic) music instrument with user interaction. (Users can control the strings. But it can only produce some noises.)

How to run it
  1. Clone the code repo to the local machine.
  2. To run the 1D shallow water simulation, enter the shallow_water folder and run the file shallow_water.pde using processing.
  3. To run the cloth and sound simulation, enter the cloth_and_audio folder and run the command "make" to build the project.
    • Run './hw2' to simulate the cloth. You can use arrow buttons to move the cube.
    • Run './audio' to produce some noises. Press button number 1 to pluck the first string and 2 for the second string.
